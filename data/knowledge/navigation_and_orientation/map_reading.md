STEP-BY-STEP ACTIONS:

1. IDENTIFY MAP TYPE (DO FIRST):
   - Topographic map: Shows elevation and terrain features
   - Road map: Shows roads and populated areas
   - Satellite map: Shows aerial view
   - Specialized map: Thematic maps for specific purposes
   - Understand map purpose: What is map designed for?
   - Understand map limitations: What does map not show?
   - Do not assume all maps are same: Different maps have different information
   - Do not use wrong map type: Wrong map type may not have needed information

2. UNDERSTAND MAP SCALE:
   - Find scale indicator: Usually in map legend or corner
   - Understand scale ratio: 1:24,000 means 1 inch = 24,000 inches real
   - Understand distance: How to measure distance on map
   - Understand area: How map area relates to real area
   - Practice measuring: Measure known distances to verify understanding
   - Do not ignore scale: Scale is essential for navigation
   - Do not assume scale: Always check scale indicator
   - Do not use wrong scale: Wrong scale causes navigation errors

3. UNDERSTAND MAP SYMBOLS:
   - Find legend: Map legend explains symbols
   - Learn common symbols: Roads, water, elevation, structures
   - Learn elevation symbols: Contour lines, elevation markers
   - Learn water symbols: Rivers, lakes, streams, marshes
   - Learn vegetation symbols: Forests, fields, clearings
   - Learn structure symbols: Buildings, bridges, towers
   - Learn road symbols: Highways, roads, trails, paths
   - Learn hazard symbols: Cliffs, swamps, dangerous areas
   - Do not ignore legend: Legend explains map symbols
   - Do not assume symbols: Always check legend

4. ORIENT MAP:
   - Align map with terrain: Map north should match real north
   - Use compass: Compass helps orient map accurately
   - Use terrain features: Match map features to real terrain
   - Use sun: Sun position helps orient map (if time known)
   - Verify orientation: Check that map matches terrain
   - Do not use unoriented map: Unoriented map causes errors
   - Do not assume orientation: Always verify map orientation
   - Do not ignore terrain: Terrain helps verify orientation

5. IDENTIFY YOUR LOCATION:
   - Look for landmarks: Identify visible landmarks
   - Match landmarks to map: Find landmarks on map
   - Use terrain features: Match terrain to map contours
   - Use intersection method: Where do features intersect?
   - Use compass bearing: Take bearings to known features
   - Verify location: Use multiple methods to confirm
   - Do not assume location: Always verify your position
   - Do not ignore landmarks: Landmarks help identify location

6. PLAN ROUTE:
   - Identify destination: Where are you going?
   - Identify starting point: Where are you starting?
   - Identify route options: What routes are available?
   - Evaluate routes: Distance, difficulty, hazards, resources
   - Choose route: Select best route for conditions
   - Identify waypoints: Landmarks along route
   - Identify hazards: Dangers along route
   - Identify resources: Water, shelter, food along route
   - Do not plan without understanding map: Understand map first
   - Do not ignore hazards: Hazards can be dangerous

7. FOLLOW ROUTE:
   - Keep map oriented: Maintain map orientation while moving
   - Follow planned route: Stay on planned path
   - Check position regularly: Verify you are on route
   - Use waypoints: Confirm you pass planned waypoints
   - Use terrain association: Match terrain to map
   - Adjust if needed: Change route if conditions require
   - Do not ignore map: Continuously refer to map
   - Do not assume you are on route: Verify regularly

8. VERIFY POSITION:
   - Check landmarks: Do landmarks match map?
   - Check terrain: Does terrain match map?
   - Check distance: Have you traveled expected distance?
   - Check direction: Are you traveling in correct direction?
   - Use multiple methods: Combine methods for accuracy
   - Do not assume position: Always verify
   - Do not ignore discrepancies: If map does not match terrain, reassess
   - Do not continue if lost: Stop and reorient if uncertain

FOR SPECIFIC MAP TYPES:

TOPOGRAPHIC MAPS:
- Understand contour lines: Lines show elevation
- Understand elevation: Higher numbers mean higher elevation
- Understand terrain: Steep slopes, valleys, ridges
- Use elevation: Elevation helps identify location
- Do not ignore elevation: Elevation is key feature
- Do not confuse contour lines: Closer lines mean steeper slope

ROAD MAPS:
- Understand road types: Highways, roads, streets
- Understand road numbers: Road numbers help navigation
- Understand populated areas: Cities, towns, villages
- Use roads: Roads are primary navigation features
- Do not ignore road numbers: Road numbers help navigation
- Do not assume all roads are shown: Some roads may not be on map

SATELLITE MAPS:
- Understand aerial view: View from above
- Understand features: Can see actual terrain
- Understand scale: May be different from other maps
- Use visual features: Match visual features to terrain
- Do not ignore scale: Scale may be different
- Do not assume features are current: Features may have changed
