STEP-BY-STEP ACTIONS:

1. ASSESS TOOL CONDITION (DO FIRST):
   - Inspect tool: Look for damage, wear, rust, problems
   - Test function: Does tool work as intended?
   - Identify problems: What needs maintenance?
   - Prioritize problems: What is most urgent?
   - Assess maintenance needs: What maintenance is required?
   - Assess maintenance feasibility: Can maintenance be done?
   - Do not ignore problems: Problems worsen if ignored
   - Do not delay urgent maintenance: Urgent problems need immediate attention

2. GATHER MAINTENANCE SUPPLIES:
   - Identify needed supplies: What supplies are required?
   - Identify available supplies: What supplies do you have?
   - Find alternatives: Can other supplies work?
   - Prepare supplies: Cut, measure, prepare as needed
   - Do not start without supplies: Incomplete maintenance fails
   - Do not use inappropriate supplies: Wrong supplies can damage tools
   - Do not waste supplies: Supplies may be limited

3. PLAN MAINTENANCE:
   - Understand problem: What is wrong? Why did it happen?
   - Plan solution: How will maintenance fix problem?
   - Plan steps: What are maintenance steps in order?
   - Plan supplies: How will supplies be used?
   - Plan tools: What tools are needed for maintenance?
   - Plan testing: How will tool be tested after maintenance?
   - Plan storage: How will tool be stored after maintenance?
   - Do not maintain without plan: Unplanned maintenance fails
   - Do not skip planning: Planning prevents mistakes

4. CLEAN TOOLS PROPERLY:
   - Remove dirt: Dirt causes wear and rust
   - Remove rust: Rust weakens tools
   - Remove debris: Debris can cause problems
   - Use appropriate cleaners: Right cleaner for material
   - Dry thoroughly: Moisture causes rust
   - Do not use harsh chemicals: Can damage tools
   - Do not leave tools wet: Causes rust
   - Do not skip cleaning: Clean tools last longer

5. LUBRICATE TOOLS PROPERLY:
   - Identify lubrication needs: What needs lubrication?
   - Use appropriate lubricant: Right lubricant for application
   - Apply correctly: Proper application method
   - Remove excess: Excess lubricant attracts dirt
   - Do not over-lubricate: Excess causes problems
   - Do not use wrong lubricant: Wrong lubricant can damage
   - Do not skip lubrication: Lubrication prevents wear

6. SHARPEN CUTTING TOOLS:
   - Assess sharpness: Is tool sharp enough?
   - Use appropriate sharpener: Right sharpener for tool
   - Use proper technique: Correct sharpening method
   - Maintain angle: Correct angle for tool
   - Test sharpness: Verify tool is sharp
   - Do not over-sharpen: Can damage tool
   - Do not use wrong sharpener: Wrong sharpener can damage
   - Do not skip sharpening: Dull tools are dangerous

7. REPAIR DAMAGE:
   - Assess damage: What is damaged? Can it be repaired?
   - Plan repair: How will damage be repaired?
   - Use appropriate materials: Right materials for repair
   - Use proper technique: Correct repair method
   - Test repair: Verify repair works
   - Do not repair if too damaged: Some damage cannot be repaired
   - Do not use inappropriate materials: Wrong materials fail
   - Do not skip repair: Damage worsens if ignored

8. REPLACE WORN PARTS:
   - Identify worn parts: What parts are worn?
   - Assess replacement needs: Do parts need replacement?
   - Find replacement parts: What parts are available?
   - Install correctly: Proper installation method
   - Test function: Verify tool works with new parts
   - Do not use worn parts: Worn parts cause failure
   - Do not skip replacement: Worn parts cause problems
   - Do not install incorrectly: Incorrect installation fails

9. STORE TOOLS PROPERLY:
   - Clean before storage: Clean tools before storing
   - Protect from moisture: Moisture causes rust
   - Protect from sun: Sun damages some tools
   - Protect from chemicals: Chemicals can damage tools
   - Protect from damage: Store in safe location
   - Use appropriate storage: Right storage for tool
   - Do not store dirty tools: Dirt causes damage
   - Do not store wet tools: Causes rust
   - Do not store in harsh conditions: Harsh conditions damage tools

    - Regular lubrication: Lubricate tools regularly

FOR SPECIFIC TOOL TYPES:

CUTTING TOOLS:
- Keep sharp: Sharp tools are safer and more effective
- Clean after use: Prevents rust and maintains function
- Oil if storing: Prevents rust
- Store safely: Protected edges, dry location
- Do not neglect sharpening: Dull tools are dangerous
- Do not store wet: Causes rust

STRIKING TOOLS:
- Check handle: Secure, not cracked or loose
- Check head: Secure, not loose or damaged
- Clean after use: Prevents rust
- Store safely: Protected, dry location
- Do not use if handle is loose: Loose handle is dangerous
- Do not use if head is loose: Loose head is dangerous

METAL TOOLS:
- Prevent rust: Clean, dry, oil if storing
- Remove rust: If rust appears, remove immediately
- Protect from moisture: Moisture causes rust
- Protect from chemicals: Chemicals can damage
- Do not store wet: Causes rust
- Do not ignore rust: Rust weakens tools

WOODEN HANDLES:
- Keep dry: Moisture causes rot and warping
- Oil occasionally: Prevents drying and cracking
- Check for cracks: Cracks weaken handles
- Repair or replace if damaged: Damaged handles are dangerous
- Do not store wet: Causes rot
- Do not ignore cracks: Cracks worsen
