IMMEDIATE PRIORITIES:
1. Assess tool condition: Is it safe to use?
2. Assess tool appropriateness: Is this the right tool for the job?
3. Assess your condition: Are you capable of safe tool use?
4. Assess environment: Are conditions safe for tool use?
5. Plan tool use: What could go wrong?
6. Maintain awareness: Continuous attention to tool and surroundings
7. Use proper technique: Correct method reduces injury risk
8. Have first aid ready: Be prepared for injuries

IMMEDIATE DANGERS:
- Cuts and lacerations: Sharp tools can cause severe cuts
- Crush injuries: Heavy tools or falling tools can crush body parts
- Puncture wounds: Pointed tools can penetrate body
- Eye injuries: Flying debris, tool parts can injure eyes
- Fractures: Impact from tools can break bones
- Amputations: Severe cuts can sever digits or limbs
- Infection: Open wounds from tools are infection risks
- Tool breakage: Broken tools can cause injury
- Loss of control: Slippery grip, fatigue cause accidents
- Injury to others: Tools can injure people nearby

STEP-BY-STEP ACTIONS:

1. SELECT APPROPRIATE TOOL (DO FIRST):
   - Match tool to task: Right tool for right job
   - Assess tool condition: Is tool in good condition?
   - Assess tool size: Is tool appropriate size for task?
   - Assess tool capability: Can tool actually do the job?
   - Assess tool safety: Does tool have safety features?
   - Do not use wrong tool: Wrong tool causes accidents
   - Do not use damaged tool: Damaged tools are dangerous
   - Do not use tool that is too small or too large: Size matters
   - Do not use tool beyond its capability: Overloading causes failure

2. INSPECT TOOL BEFORE USE:
   - Check for damage: Cracks, bends, breaks, wear
   - Check for loose parts: Handles, heads, moving parts
   - Check for sharp edges: Sharp edges can cut
   - Check for rust: Rust weakens tools and causes tetanus risk
   - Check function: Does tool work as intended?
   - Check safety features: Guards, locks, safety mechanisms
   - If tool is damaged: Do not use, repair or find alternative
   - If tool is unsafe: Do not use, safety is priority
   - Do not use damaged tools: They are more dangerous

3. PREPARE WORK AREA:
   - Clear area: Remove obstacles that could cause accidents
   - Ensure adequate light: You must see what you are doing
   - Ensure stable surface: Unstable surface causes loss of control
   - Ensure adequate space: Room to work without hitting obstacles
   - Secure materials: Materials must be held securely
   - Position yourself: Stable stance, balanced position
   - Clear path for tool: Where will tool go if it slips?
   - Remove distractions: Focus is essential for safety
   - Have first aid ready: Be prepared for injuries
   - Do not work in cramped or cluttered areas

4. USE PROPER TECHNIQUE:
   - Use tool as designed: Follow intended use
   - Maintain secure grip: Firm but not crushing grip
   - Keep hands away from working parts: Fingers away from blades, impact points
   - Use controlled force: Appropriate force, not excessive
   - Support materials: Materials must be secured
   - Use cutting board or work surface: Protects tool and surface
   - Work away from body: Tool should not point toward you
   - Work away from others: Ensure no one is in tool path
   - Do not force tool: Forcing causes loss of control
   - Do not use tool inappropriately: Wrong use causes accidents

5. MAINTAIN TOOL AWARENESS:
   - Always know where tool is: Continuous awareness of tool position
   - Always know where tool is going: Plan tool path
   - Always know what is behind work: What will tool hit if it goes through?
   - Keep tool in sight: Do not work where you cannot see
   - Watch for tool movement: Tools can move unexpectedly
   - Watch for material movement: Materials can shift
   - Do not work blindly: You must see what you are doing
   - Do not work when distracted: Focus is essential
   - Do not work when tired: Fatigue causes mistakes

6. PROTECT YOURSELF:
   - Use eye protection: If available, protect eyes from debris
   - Use hand protection: Gloves if appropriate, but ensure grip
   - Use appropriate clothing: Loose clothing can catch in tools
   - Keep hands dry: Wet hands cause slippage
   - Keep hands clean: Dirt or oil reduces grip
   - Maintain balance: Stable stance prevents falls
   - Do not wear loose clothing: Can catch in tools
   - Do not wear jewelry: Can catch in tools
   - Do not work with wet or oily hands: Causes loss of control

7. HANDLE TOOLS SAFELY WHEN NOT IN USE:
   - Store tools safely: Secure location, protected edges
   - Pass tools safely: Handle first, working end away
   - Do not leave tools unattended: Others may misuse
   - Do not carry tools unsafely: Use sheaths, cases, or proper carry
   - Do not play with tools: Tools are not toys
   - Do not throw tools: Throwing tools is dangerous
   - Do not use tools for inappropriate tasks: Use right tool for job
   - Do not leave tools where they can fall: Falling tools cause injury

8. RESPOND TO PROBLEMS SAFELY:
   - If tool slips: Stop immediately, assess for injury
   - If material moves: Stop, reposition, secure material
   - If grip becomes slippery: Stop, clean hands and tool
   - If tool becomes stuck: Do not force, work tool free carefully
   - If tool breaks: Stop immediately, assess for injury
   - If you become distracted: Stop, refocus, then continue
   - If you become tired: Stop, rest, then continue
   - If conditions become unsafe: Stop, address conditions, then continue
   - Do not continue if conditions are unsafe
   - Do not ignore problems: Address immediately

9. MAINTAIN TOOLS PROPERLY:
   - Clean tools after use: Prevents rust and maintains function
   - Oil tools if storing: Prevents rust
   - Check tools regularly: Ensure they remain in good condition
   - Repair damage: Fix problems before they worsen
   - Sharpen cutting tools: Sharp tools are safer than dull tools
   - Store properly: Dry location, protected edges
   - Do not neglect maintenance: Poor maintenance causes accidents
   - Do not use damaged tools: Repair or replace

FOR SPECIFIC TOOL TYPES:

HAMMERS AND STRIKING TOOLS:
- Check handle: Secure, not cracked or loose
- Check head: Secure, not loose or damaged
- Use appropriate size: Too large or too small is dangerous
- Strike squarely: Angled strikes cause tool to slip
- Watch for flying debris: Debris can injure eyes
- Keep hands away from strike area: Fingers can be crushed
- Do not use damaged hammer: Loose head can fly off
- Do not strike hardened surfaces: Can cause tool to break

SAWS AND CUTTING TOOLS:
- Check blade: Sharp, not bent or damaged
- Check teeth: Intact, not missing or damaged
- Secure material: Material must not move
- Cut away from body: Never toward yourself
- Use appropriate saw: Right saw for material
- Let saw do the work: Do not force
- Watch for binding: Blade can bind and cause kickback
- Do not force saw: Forcing causes loss of control
- Do not use damaged blade: Damaged blades are dangerous

WRENCHES AND TURNING TOOLS:
- Check tool: Not cracked, bent, or damaged
- Use correct size: Wrong size causes slippage
- Pull toward you: Pushing away causes loss of control
- Use steady force: Jerky force causes slippage
- Watch for sudden release: Fasteners can break suddenly
- Keep hands away from moving parts: Fingers can be pinched
- Do not use wrong size: Causes tool to slip
- Do not use excessive force: Can break fastener or tool

SCREWDRIVERS AND PUSHING TOOLS:
- Check tip: Not worn, damaged, or wrong type
- Use correct type: Right type for fastener
- Use correct size: Right size for fastener
- Apply steady pressure: Maintain control
- Keep hands away from tip: Tip can slip
- Do not use as pry bar: Not designed for prying
- Do not use wrong type: Causes damage and slippage
- Do not use excessive force: Can break tool or fastener

YOUR MENTAL STATE:

- Clarity: Can you think clearly?
- Stress: Is stress affecting your control?
- Distraction: Are you distracted?
- If mental state is compromised → do not use tools

TOOL CONDITION:
- Integrity: Is tool intact? Not damaged?
- Function: Does tool work as intended?
- Safety features: Are safety features working?

WORK CONDITIONS:

- Materials: Are materials secured?
- These signs indicate if conditions are safe
