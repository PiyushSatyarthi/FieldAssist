   - Trauma: Is trauma causing deterioration?
   - Physical illness: Is physical illness causing deterioration?
   - Do not ignore causes: Causes must be addressed
   - Do not assume causes: Verify causes

4. ADDRESS CAUSES:
   - Reduce stress: Reduce stress if possible
   - Address isolation: Address isolation if possible
   - Improve sleep: Improve sleep if possible
   - Address trauma: Address trauma if possible
   - Treat physical illness: Treat physical illness if possible
   - Do not ignore causes: Causes must be addressed
   - Do not delay treatment: Delay worsens deterioration

   - Emotional support: Provide emotional support
   - Practical support: Provide practical support

   - Maintain routine: Maintain routine if possible
   - Maintain self-care: Maintain self-care if possible
